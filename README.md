# Rust-timetable-calendar
This is a Rust-powered calendar app designed to help users effortlessly plan, organize, and schedule their daily activities.
It is a simple, fast, and efficient Rust-powered timetable/calendar app that helps users plan their day, schedule activities, and stay organized.

This project was built to demonstrate how Rust can be used to create lightweight command-line tools that handle user input, scheduling, and time-management tasks with reliability and speed.
